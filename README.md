# Chess Game WebApp

A browser-based chess game built with HTML, CSS, and JavaScript.

## Features
- Play chess against another player on the same computer
- Move validation for each piece (Pawn, Rook, Knight, Bishop, Queen, King)
- Detect check and checkmate conditions
- Reset game anytime

## Technologies
- HTML
- CSS
- JavaScript

## How to Run
1. Clone or download this repository
2. Open `index.html` in your web browser
3. Start playing!
